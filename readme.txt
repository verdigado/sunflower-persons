Custom Post Type "person" und Gutenberg-Block zur Anzeige einer Person oder einer Liste von Personen.

== Beschreibung ==
- CPT: person
- Block: sunflower-person/person

== Installation ==
1. ZIP hochladen und aktivieren.
2. Unter "Personen" Einträge anlegen.
3. Im Block-Editor "Person(en)" Block einfügen und optional eine Person wählen.
